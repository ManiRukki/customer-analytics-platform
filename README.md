# Customer Analytics Platform

**Problem:** Stakeholders lacked visibility into churn and revenue drivers.

**Solution:** Built interactive analytics platform with Python ETL and Streamlit dashboard.

**Outcome:** Sales team reduced churn by 10%.

## Tech Stack
- Python, Pandas, SQLAlchemy
- Streamlit, Plotly
- SQLite/Postgres

## Quick Start
```bash
pip install -r requirements.txt
python etl/load_data.py
streamlit run app/streamlit_app.py
```
